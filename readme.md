# Laravel-Vue-JWT SPA

## Features

- Laravel 6.0
- Vue + VueRouter + Vuex + VueI18n + ESlint
- Authentication with JWT

## Installation

- git clone or download
- `php artisan key:generate`
- `php artisan jwt:secret`
- `php artisan migrate`
- `npm install`

## Usage

#### Development

```bash
# build and watch
npm run watch

# serve with hot reloading
npm run hot
```

#### Production

```bash
npm run production
```
